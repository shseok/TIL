# RegExp
RegExp 생성자는 패턴을 사용해 텍스트를 판별할 때 사용한다

### 정규식을 만드는 방법

1.정규식 리터럴
```
    var re = /ab+c/;
```
2. 객체의 생성자 함수 호출
```
    var re = new RegExp("ab+c");
```
