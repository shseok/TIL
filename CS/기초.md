- Computer System
    - 일반적으로 4가지 요소로 구성
        - User (Human computer Interface ...)
        - Hardware (디지털 회로, 컴퓨터 구조...)
        - Software (운영체제, 데이터베이스, 시스템프로그래밍 ...)
        - Data(Infomation)
